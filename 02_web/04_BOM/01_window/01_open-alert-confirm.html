자바스크립트는 웹에서 동작하는 언어.
        js가 web에서 동작할 때 사용가능한 기능이 있다.

        window
            document object method
            - DOM (document 등등)       // 내가 만들고 있는 또는 보고있는 페이지 document를 제어할 수 있게 하는 것들
            browser object method
            - BOM (windows)             // 화면이 뜨고 있는 브라우저 자체를 제어할 수 잇도록 하는 것들            
            - js 
            
        번외
            - CSSOM                     // css를 조작할 수 있게 하는 것들

confirm

        window.confirm(message) -> 확인과 취소 버튼을 가지고 있으며 메세지를 지정할 수 있다.
        반환값이 존재(true,false)

prompt
